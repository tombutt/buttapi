Welcome to ButtAPI.


As I belive in giving credit where credit is due, this was fully inspired by one of my favorite sites, http://foaas.com.


ButtAPI is an OpenSource project which can be found on github. ButtAPI is written in node.js, so if you feel the need to contribute, it's quite simple (based on my desire to accept any contributions.)


Currently, the only result type is text/plain, but there are endless future enhancements that will be done in this space.


API


/stinks/:name

“:name stinks like a butt.”


/isa/:name

“:name is a butt.”


/suy/:name/:object

“Stick a :object up your butt :name”


/iuy/:name/:object

“Is your :object up your butt :name”
